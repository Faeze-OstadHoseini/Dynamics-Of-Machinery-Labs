# Lab 2: Gyroscope Precession Analysis

## Overview
This laboratory experiment investigates the phenomenon of gyroscopic precession. The gyroscope rotor spins at different speeds, and the resulting precession speed is measured under various masses. The theoretical precession speed is calculated and compared with experimental results.

## Theory

### Gyroscopic Precession
When a torque is applied to a spinning gyroscope, it responds with a precession motion instead of falling over. The relationship between precession speed (wp), rotor speed (ws), and applied torque (T) is:

T = I * ws * wp

Where:
T = Applied torque (N.m)
I = Moment of inertia (kg.m^2)
ws = Rotor speed (rad/s)
wp = Precession speed (rad/s)

### Applied Torque
The torque is generated by a hanging mass:

T = m * g * R

Where:
m = Hanging mass (kg)
g = Gravitational acceleration (9.81 m/s^2)
R = Radius of the pulley (0.094 m)

### Theoretical Precession Speed
From the two equations above:

wp_theo = T / (I * ws)

## Calculations Performed

For each case, the script performs the following calculations:

1. Force: F = m * g
2. Torque: T = F * R
3. Convert rotor speed from rpm to rad/s: ws_rad = ws * (2 * pi / 60)
4. Theoretical precession speed (rad/s): wp_theo_rad = T / (I * ws_rad)
5. Convert theoretical precession to rpm: wp_theo = wp_theo_rad * (60 / (2 * pi))
6. Error percentage: Error = (abs(wp_theo - wp_exp) / wp_theo) * 100

## Observations

1. As rotor speed increases, precession speed decreases (inverse relationship)
2. Higher mass results in higher precession speed
3. Theoretical and experimental values show some deviation due to:
   - Friction in the system
   - Measurement errors
   - Assumptions in the theoretical model
